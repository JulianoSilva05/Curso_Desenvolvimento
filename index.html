Tema 01: Conhecendo a Linguagem PHP

Uma Introdução ao Desenvolvimento Back-End

O que é PHP?

A linguagem PHP (Hypertext Preprocessor), criada por Rasmus Lerdorf em 1994, é uma linguagem de código aberto voltada para o desenvolvimento web. O código é interpretado no lado do servidor, que processa a lógica e envia o resultado como uma página HTML para o navegador do usuário.

Seu Primeiro Código

O código PHP é escrito dentro das tags <?php ... ?> e os arquivos são salvos com a extensão .php. O comando echo é usado para exibir texto.


<html>
  <body>
    <h1>Meu primeiro site com PHP</h1>
    <?php
      echo "Olá, Mundo!";
    ?>
  </body>
</html>

Tema 02: Variáveis e Tipos de Dados

Armazenando e Manipulando Informações

Tipos de Dados Primitivos

Uma variável é um local temporário para guardar dados. Em PHP, elas começam com $ e guardam diferentes tipos de dados.


// String (Texto)
$nome = "Anselmo";

// Integer (Inteiro)
$idade = 38;

// Float (Ponto Flutuante / Decimal)
$salario = 3000.50;

// Boolean (Verdadeiro ou Falso)
$casado = true;

Exibição e Concatenação

Use echo para exibir dados. Use o ponto . para unir (concatenar) strings e variáveis.


$aluno = "Carol almeida";
$curso = "Técnico em informática";

// Exibindo os dados concatenados
echo "Aluno: " . $aluno . " | Curso: " . $curso;

Tema 03: Operadores

Realizando Operações e Comparações

Operadores Aritméticos

São responsáveis por fazer os cálculos tradicionais usados na matemática.


$v1 = 15;
$v2 = 12;

$soma = $v1 + $v2; // Adição
$subtracao = $v1 - $v2; // Subtração
$multiplicacao = $v1 * $v2; // Multiplicação
$divisao = $v1 / $v2; // Divisão
$modulo = $v1 % $v2; // Resto da divisão

echo "A soma é: " . $soma;

Operadores de Comparação

Têm a finalidade de comparar valores e dados na programação. O resultado é sempre um booleano (`true` ou `false`).


$b = 10;
$c = "10";

var_dump($b == $c);   // bool(true) - '==' Igualdade, compara apenas o valor
var_dump($b === $c);  // bool(false) - '===' Idêntico, compara valor E tipo
var_dump($b != $c);   // bool(false) - '!=' Não igual
var_dump($b > 5);     // bool(true) - '>' Maior que

Tema 04: Estrutura de Controle

Tomando Decisões e Repetindo Tarefas

Estruturas Condicionais

Permitem que o programa execute blocos de código diferentes com base em condições.


$hora = date('H'); // Pega a hora atual do servidor

if ($hora < 12) { // Caso a expressão seja verdadeira (true), a programação segue um caminho
  echo "Acorda, Bom dia!";
} elseif ($hora < 18) { // Combinação entre ELSE e IF
  echo "Hora do café, Boa tarde!";
} else { // Caso seja falso, determina-se uma sequência lógica
  echo "Tomar banho, Boa noite";
}

Laços de Repetição (Loops)

Usados para executar o mesmo bloco de código várias vezes.


// 'for': executa um número definido de vezes, com início e fim específico.
for ($i = 1; $i <= 5; $i++) {
  echo $i . " ";
}

// 'while': executa enquanto uma condição for verdadeira. Sabe-se o início, mas não o fim.
$num = 1;
while ($num <= 5) {
  echo $num . " ";
  $num++;
}

// 'foreach': ideal para percorrer arrays.
$frutas = array("Maçã", "Banana", "Laranja");
foreach ($frutas as $fruta) {
  echo $fruta . " ";
}

Tema 05: Conceito sobre Array

Estruturando Dados de Forma Eficiente

Tipos de Array

Um Array é uma estrutura que armazena múltiplos valores em uma única variável.

  • Arrays Indexados: Usam índices numéricos para acessar os elementos.
  • Arrays Associativos: Usam chaves alfanuméricas (nomes) para acessar os elementos.
  • Arrays Multidimensionais: Contêm outros arrays como elementos, formando uma matriz.

// Array Indexado
$frutas = array("maçã", "banana", "laranja");
echo $frutas[0]; // imprime maçã

// Array Associativo
$pessoa = array("nome" => "João", "idade" => 25);
echo $pessoa["nome"]; // imprime João

Tema 06: Método de envio com GET e POST

Enviando Dados de Formulários

Métodos GET e POST

Para enviar dados de um formulário HTML para o PHP, usamos os métodos GET e POST.

  • Método GET: Os dados são enviados pela URL. É o método padrão, mas não suporta grande volume de dados e não é seguro para informações sensíveis.
  • Método POST: Os dados não são passados pela URL, mas sim no corpo da requisição. É considerado mais seguro e recomendado para criar um CRUD.

// Formulário HTML
<form action="pag.php" method="post">
  Nome: <input type="text" name="cxnome">
  <input type="submit" value="Enviar">
</form>

// Recebendo dados com POST em pag.php
$nome = $_POST["cxnome"];
echo "Nome: " . $nome;

Tema 07: Tipos de Funções

Reutilizando Blocos de Código

O que são Funções?

Funções são blocos de código que podem ser reutilizados em diferentes partes do programa, evitando repetições e facilitando a manutenção. Existem as funções nativas (da própria linguagem) e as criadas pelo desenvolvedor.


// Para criar uma função, usamos a palavra-chave function
function soma($a, $b) { // $a e $b são parâmetros
  $resultado = $a + $b;
  return $resultado; // 'return' envia um valor de volta
}

// Para usar a função, basta chamá-la pelo nome
echo soma(3, 5); // imprime 8

Tema 08: Tipos de Cookies e Sessões

Gerenciando Dados do Usuário

Cookies vs. Sessões

  • Cookies: São pequenos arquivos gerados pelo servidor e armazenados no navegador do cliente. Eles guardam informações como fotos, vídeos e textos para acelerar o acesso futuro.
  • Sessões (Session): Armazenam informações sobre um único usuário no servidor e ficam disponíveis para todas as páginas da aplicação. São muito úteis para segurança e para funcionalidades como carrinhos de compra.

// Para usar sessões, inicie com session_start() em todas as páginas
session_start();

// Armazenando dados em uma variável de sessão
$_SESSION["usuario"] = "Anselmo Paulo";

echo "Usuário logado: " . $_SESSION["usuario"];

// Para finalizar a sessão (logout)
session_destroy();

Tema 09: Conexão com o banco de dados

Persistindo Dados com MySQL

Conectando com MySQL e CRUD

A conexão com o banco de dados é um ponto central de qualquer linguagem de programação. CRUD é um acrônimo para as quatro operações básicas de um banco de dados: Create (Criar), Read (Ler), Update (Atualizar) e Delete (Excluir).


// 1. Arquivo de Conexão (conexao.php)
$server = "localhost";
$user = "root";
$password = "";
$dbname = "bdloja2022";
$conn = mysqli_connect($server, $user, $password, $dbname);

// 2. Inserindo dados (Create)
include_once 'conexao.php';
$nome = $_POST['cnome'];
$sql = "insert into cliente (nome) values ('$nome')";
$query = mysqli_query($conn, $sql);

// 3. Consultando dados (Read)
$consultar = "select * from tbcliente";
$executar = mysqli_query($conn, $consultar);
while ($linha = mysqli_fetch_array($executar)) {
    echo "Nome: " . $linha["nome"];
}

Tema 10: Introdução ao conceito POO e classes com PHP

Organizando Código com Objetos

Classes e Objetos

A Programação Orientada a Objetos (POO) é uma maneira de programar de forma organizada, segura e reutilizável.

  • Classe: É uma abstração, um modelo para criar objetos. Ela agrupa comportamentos e características similares.
  • Objeto: É a instância de uma classe, ou seja, a sua representação física que ocupa espaço na memória. Para instanciar uma classe, usa-se o comando `new`.

// Classe é uma entidade teórica, uma organização
class Carro {
    public $fabricante; // 'public' define a visibilidade
}

// Quando instanciamos, $car1 se torna um objeto
$car1 = new Carro();
$car1->fabricante = 'VW'; // Atribuindo um valor à propriedade
