Tema 01: Conhecendo a Linguagem PHP

Uma Introdução ao Desenvolvimento Back-End

O que é PHP?

[cite_start]

A linguagem PHP (Hypertext Preprocessor), criada por Rasmus Lerdorf em 1994, é uma linguagem de código aberto voltada para o desenvolvimento web[cite: 282]. [cite_start]O código é interpretado no lado do servidor, que processa a lógica e envia o resultado como uma página HTML para o navegador do usuário[cite: 282].

Seu Primeiro Código

[cite_start]

O código PHP é escrito dentro das tags <?php ... ?> e os arquivos são salvos com a extensão .php[cite: 442]. [cite_start]O comando echo é usado para exibir texto[cite: 627].


<html>
  <body>
    <h1>Meu primeiro site com PHP</h1>
    <?php
      echo "Olá, Mundo!";
    ?>
  </body>
</html>

Tema 04: Estrutura de Controle

Tomando Decisões e Repetindo Tarefas

Estruturas Condicionais

[cite_start]

Permitem que o programa execute blocos de código diferentes com base em condições[cite: 918, 919].


$hora = date('H'); // Pega a hora atual do servidor

[cite_start]if ($hora < 12) { // Caso a expressão seja verdadeira (true), a programação segue um caminho [cite: 922]
  echo "Acorda, Bom dia!";
[cite_start]} elseif ($hora < 18) { // Combinação entre ELSE e IF [cite: 951]
  echo "Hora do café, Boa tarde!";
[cite_start]} else { // Caso seja falso, determina-se uma sequência lógica [cite: 935]
  echo "Tomar banho, Boa noite";
}

Laços de Repetição (Loops)

[cite_start]

Usados para executar o mesmo bloco de código várias vezes[cite: 990].


[cite_start]// 'for': executa um número definido de vezes, com início e fim específico[cite: 1038, 1039].
for ($i = 1; $i <= 5; $i++) {
  echo $i . " ";
}

// 'while': executa enquanto uma condição for verdadeira. [cite_start]Sabe-se o início, mas não o fim[cite: 994].
$num = 1;
while ($num <= 5) {
  echo $num . " ";
  $num++;
}

[cite_start]// 'foreach': ideal para percorrer arrays[cite: 1050].
$frutas = array("Maçã", "Banana", "Laranja");
foreach ($frutas as $fruta) {
  echo $fruta . " ";
}

Tema 05: Conceito sobre Array

Tipos de Array

[cite_start]

Um Array é uma estrutura que armazena múltiplos valores em uma única variável[cite: 1130, 1134].

  • Arrays Indexados: Usam índices numéricos para acessar os elementos[cite: 1141].

Tema 06: Método de envio com GET e POST

Enviando Dados de Formulários

Métodos GET e POST

[cite_start]

Para enviar dados de um formulário HTML para o PHP, usamos os métodos GET e POST[cite: 1348].

    [cite_start]
  • Método GET: Os dados são enviados pela URL[cite: 1358]. [cite_start]É o método padrão, mas não suporta grande volume de dados e não é seguro para informações sensíveis[cite: 1358, 1359].
    • [cite_start]
  • Método POST: Os dados não são passados pela URL, mas sim no corpo da requisição[cite: 1397]. [cite_start]É considerado mais seguro e recomendado para criar um CRUD[cite: 1398, 1400].

[cite_start]// Formulário HTML [cite: 1402]
<form action="pag.php" method="post">
  Nome: <input type="text" name="cxnome">
  <input type="submit" value="Enviar">
</form>

[cite_start]// Recebendo dados com POST em pag.php [cite: 1419]
$nome = $_POST["cxnome"];
echo "Nome: " . $nome;

Tema 08: Tipos de Cookies e Sessões

Gerenciando Dados do Usuário

Cookies vs. Sessões

    [cite_start]
  • Cookies: São pequenos arquivos gerados pelo servidor e armazenados no navegador do cliente[cite: 1686, 1698]. [cite_start]Eles guardam informações como fotos, vídeos e textos para acelerar o acesso futuro[cite: 1687].
    • [cite_start]
  • Sessões (Session): Armazenam informações sobre um único usuário no servidor e ficam disponíveis para todas as páginas da aplicação[cite: 1688, 1694]. [cite_start]São muito úteis para segurança e para funcionalidades como carrinhos de compra[cite: 1695].

[cite_start]// Para usar sessões, inicie com session_start() em todas as páginas [cite: 1745]
session_start();

[cite_start]// Armazenando dados em uma variável de sessão [cite: 1742]
$_SESSION["usuario"] = "Anselmo Paulo";

echo "Usuário logado: " . $_SESSION["usuario"];

[cite_start]// Para finalizar a sessão (logout) [cite: 1752]
session_destroy();
